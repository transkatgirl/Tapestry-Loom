## Tapestry Weave

This crate contains [Tapestry Loom](https://github.com/transkatgirl/Tapestry-Loom)'s document format, which is a thin wrapper around the [universal-weave](../universal-weave) crate.

Unlike Tapestry Loom (which is licensed under AGPLv3), both universal-weave and tapestry-weave are licensed under the [Unlicense](./LICENSE).
